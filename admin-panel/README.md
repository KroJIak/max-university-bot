# Админ панель университета

Админ панель для управления функционалом бота и настройки University API.

## Функционал

1. **Настройка University API**
   - Указание Base URL для University API
   - Сохранение конфигурации в базу данных

2. **Управление функциями**
   - Включение/выключение функций через переключатели
   - Настройка endpoints для каждой функции
   - Автоматическое сохранение изменений

## Доступные функции

- **Логин студентов** (`students_login`) - привязка аккаунтов студентов
- **Статус студента** (`students_status`) - проверка статуса привязки
- **Техническая страница** (`students_tech`) - получение защищенных страниц
- **CRUD студентов** (`students_crud`) - полное управление данными

## Запуск

Админ панель доступна на порту 8080 (настраивается через `ADMIN_PANEL_PORT` в `.env`).

## API

Админ панель использует следующие эндпоинты Main API:

- `GET /api/v1/config/university` - получить конфигурацию
- `PUT /api/v1/config/university` - сохранить конфигурацию
