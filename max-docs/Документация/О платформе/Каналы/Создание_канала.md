# Создание каналов для бизнеса

> Подключение к платформе MAX для партнёров и сервисам платформы — чат-ботам, мини-приложениям, каналам — пока доступно для ограниченного списка юрлиц и ИП

## [](#Каналы для бизнеса)Каналы для бизнеса

С помощью каналов ваша организация может напрямую взаимодействовать с аудиторией, например публиковать новости и анонсировать акции

![](/assets/max_business_platform_channels.png) _Платформа MAX для партнёров_

Процесс создания каналов и их количество зависит от присутствия вашей организации с меткой А+ на других платформах (ВКонтакте, Одноклассники, Дзен, Telegram)

> На этой странице описано создание каналов для юрлиц и ИП. Как добавить каналы, если у вас другая организационно-правовая форма и вы автор канала с А+, читайте в [FAQ MAX](https://help.max.ru/help/channels)

### [](#Если у организации нет ресурсов с меткой А+ на других платформах)Если у организации нет ресурсов с меткой А+ на других платформах

Вы можете [создать](#%D0%9A%D0%B0%D0%BA%20%D1%81%D0%BE%D0%B7%D0%B4%D0%B0%D1%82%D1%8C%20%D0%BA%D0%B0%D0%BD%D0%B0%D0%BB%20%D1%87%D0%B5%D1%80%D0%B5%D0%B7%20%D0%BF%D0%BB%D0%B0%D1%82%D1%84%D0%BE%D1%80%D0%BC%D1%83) канал через платформу [MAX для партнёров](https://business.max.ru/self). Пока доступно создание только одного такого канала

Для удобства и безопасности мы разделили верификацию и повседневное использование канала между двумя платформами: MAX для партнёров и мессенджером MAX

На **платформе MAX для партнёров** вы [верифицируете](/docs/maxbusiness/connection#%D0%92%D0%B5%D1%80%D0%B8%D1%84%D0%B8%D0%BA%D0%B0%D1%86%D0%B8%D1%8F%20%D0%BE%D1%80%D0%B3%D0%B0%D0%BD%D0%B8%D0%B7%D0%B0%D1%86%D0%B8%D0%B8) организацию и инициируете создание канала. Только подтверждённые организации могут создать канал через платформу

**Мессенджер MAX** вы используете для ежедневной работы с каналом. Создание контента, публикация постов, управление каналом — всё это происходит непосредственно в интерфейсе мессенджера

### [](#Если у организации есть ресурсы с меткой А+ на других платформах)Если у организации есть ресурсы с меткой А+ на других платформах

Вы можете:

  * Создать каналы [через бота **Каналы в MAX**](#%D0%9A%D0%B0%D0%BA%20%D1%81%D0%BE%D0%B7%D0%B4%D0%B0%D1%82%D1%8C%20%D0%BA%D0%B0%D0%BD%D0%B0%D0%BB%20%D1%87%D0%B5%D1%80%D0%B5%D0%B7%20%D0%B1%D0%BE%D1%82%D0%B0%20%D0%9A%D0%B0%D0%BD%D0%B0%D0%BB%D1%8B%20%D0%B2%20MAX%20%D0%B4%D0%BB%D1%8F%20%D1%80%D0%B5%D1%81%D1%83%D1%80%D1%81%D0%BE%D0%B2%20%D1%81%20%D0%BC%D0%B5%D1%82%D0%BA%D0%BE%D0%B9%20%D0%90+), с такими же никами и названиями как у ваших ресурсов с меткой А+ на других платформах (ВКонтакте, Одноклассники, Дзен, Telegram). Количество каналов неограничено
  * Создать дополнительный канал [через платформу MAX для партнёров](#%D0%9A%D0%B0%D0%BA%20%D1%81%D0%BE%D0%B7%D0%B4%D0%B0%D1%82%D1%8C%20%D0%BA%D0%B0%D0%BD%D0%B0%D0%BB%20%D1%87%D0%B5%D1%80%D0%B5%D0%B7%20%D0%BF%D0%BB%D0%B0%D1%82%D1%84%D0%BE%D1%80%D0%BC%D1%83) и бота **Каналы в MAX для бизнеса**. При этом название канала вы выбираете сами, а ник генерируется автоматически. Пока доступно создание только одного такого канала

## [](#Как создать канал через платформу)Как создать канал через платформу

> Юрлицо или ИП может создать пока только один канал через платформу. Ник генерируется автоматически по шаблону `idИНН_biz`

### [](#Что потребуется)Что потребуется

  * Пройти регистрацию на платформе и верифицировать организацию. Подробнее — [в разделе «Подключение к платформе»](/docs/maxbusiness/connection)
  * Профиль в MAX с номером телефона, который совпадает с тем, что вы использовали при регистрации на платформе

### [](#Этап 1. На платформе MAX для партнёров)Этап 1. На платформе MAX для партнёров

  1. Откройте [платформу MAX для партнёров](https://business.max.ru/self)
  2. Перейдите в раздел **Канал** → **Создать канал**
  3. Отсканируйте QR-код или нажмите кнопку — вы перейдёте в бота для создания каналов в MAX

### [](#Этап 2. В мессенджере MAX)Этап 2. В мессенджере MAX

  1. В боте **Каналы в MAX для бизнеса** нажмите **Начать** , затем выберите **Создать канал**
  2. Поделитесь номером телефона. Он должен совпадать с номером, который вы используете при входе на платформу
  3. Дождитесь обработки заявки — вам придёт уведомление
  4. Введите название канала.

> Запрещено использовать ненормативную лексику и контент 18+

  5. Получите название и ник. Ник генерируется автоматически по шаблону `idИНН_biz`
  6. Нажмите **Создать канал**. Если всё в порядке, в MAX будет создан ваш канал

Готово! Теперь вы можете [добавить](/docs/channels/manage) аватар и описание канала и опубликовать первый пост

## [](#Как создать канал через бота Каналы в MAX для ресурсов с меткой А+)Как создать канал через бота **Каналы в MAX** для ресурсов с меткой А+

Если у вас есть канал или сообщество с меткой А+ на другой платформе (ВКонтакте, Одноклассники, Дзен или Telegram), вы можете создать в MAX каналы с такими же названиями и никами через бота **Каналы в МАХ**. Ограничений на количество таких каналов нет

> Ваш номер телефона в MAX должен совпадать с тем, что вы использовали для получения метки А+ в Роскомнадзоре

  1. Перейдите в чат-бот **Каналы в MAX** : [@channel_bot](https://max.ru/channel_bot) → **Начать**
  2. Нажмите **Создать канал**
  3. Выберите платформу, где у вас есть метка А+. От этого зависят дальнейшие шаги

**Если у вас есть метка А+ в ВКонтакте, ОК или Дзене:**

  1. Поделитесь номером телефона из MAX — это нужно, чтобы мы убедились, что вы автор. Номер должен совпадать с тем, что вы использовали для получения метки А+
  2. Отправьте ссылку на ваш ресурс с меткой А+
     * Для ВКонтакте: ссылку на сообщество ВКонтакте
     * Для ОК: ссылку на группу в Одноклассниках
     * Для Дзена: ссылку на канал
  3. Отправьте ссылку на регистрацию вашего ресурса в Роскомнадзоре в формате `https://www.gosuslugi.ru/snet/...` или `https://knd.gov.ru/license?id=...`   
После проверки бот пришлёт вам соответствующее уведомление
  4. Нажмите **Создать канал**. Если всё в порядке, MAX пришлёт уведомление с ником и названием канала:
     * Для ВКонтакте ник канала — часть после домена `vk.com/`
     * Для ОК ник канала — в формате `group` \+ цифровой идентификатор
     * Для Дзена ник канала — часть после `https://dzen.ru/`

**Если у вас метка А+ в Telegram:**

  1. Отправьте ссылку на регистрацию вашего канала в Роскомнадзоре в формате `https://www.gosuslugi.ru/snet/...` или `https://knd.gov.ru/license?id=...`
  2. Вам придёт уникальный хештег — добавьте его в описание вашего канала в Telegram. После создания канала хештег можно будет удалить
  3. В боте **Каналы в МАХ** нажмите **Проверить**. После проверки бот пришлёт вам соответствующее уведомление
  4. Нажмите **Создать канал**. Если всё в порядке, MAX пришлёт уведомление с ником и названием канала:
     * Для Telegram ник канала — часть после `t.me/`

Готово! Теперь вы можете [добавить](/docs/channels/manage) аватар и описание канала и опубликовать первый пост
